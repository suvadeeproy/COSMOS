# COSMOS — Physics-Plus Build (v4)

A rigorous, interactive scientific-ML workbench with **live physics commentary** and **professional About** page.

**Modules**
1. **Exoplanets** — real or synthetic light curves; BLS-like period scan; box-fit; residuals; SNR; **live notes**.
2. **Landscape** — GBM regression/classification; conformal intervals; permutation FI; partial dependence; **live UQ/coverage**.
3. **AdS/QCD** — soft-/hard-/hybrid-wall; FD eigen-solve; WKB; orthonormality; virial; convergence; **live theory–numerics delta**.
4. **Benchmarks** — static CSV (robust) **+ live session metrics** aggregated from all tabs.

## Local run
```bash
pip install -r requirements.txt
streamlit run app.py
```
