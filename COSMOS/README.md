# COSMOS — Physics-Plus Build (v4)

A rigorous, interactive scientific‑ML workbench with **live physics commentary** and **session telemetry**.

**Modules**
1. **Exoplanets** — real/synthetic light curves; period scan; box‑fit; residuals; SNR; physics explainer.
2. **Landscape** — GBM regression/classification; conformal intervals; permutation FI; partial dependence; explainer.
3. **AdS/QCD** — soft‑/hard‑/hybrid‑wall; FD eigen‑solve; WKB; orthonormality; virial; convergence; explainer.
4. **Benchmarks** — deterministic metrics + **Live Telemetry** (this‑session metrics & plots).
5. **About** — professional overview, significance, reproducibility/ethics, citations.

## Local run
```bash
pip install -r requirements.txt
streamlit run app.py
```
