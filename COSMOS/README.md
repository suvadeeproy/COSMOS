# COSMOS — Scientific ML Workbench (Ready Build v2)

Hardened Streamlit app with zero-cost deploy and **no fragile deps**.
- Exoplanets: optional `lightkurve` (real data) with **safe fallback**.
- Landscape: GBM regression/classification with **conformal intervals** (MAPIE) and robust labels.
- AdS/QCD: finite-difference Sturm–Liouville solver with **convergence**.
- Benchmarks: local CSV with **robust path resolution** and **auto-regeneration** if missing.

## Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
