# COSMOS — Physics-Plus Build (v3)

A rigorous, interactive scientific‑ML workbench with deeper physics controls.

**Modules**
1. **Exoplanets** — real or synthetic light curves; BLS; box-fit; residuals; SNR; export.
2. **Landscape** — GBM regression/classification; conformal intervals; permutation FI; partial dependence.
3. **AdS/QCD** — soft‑wall / hard‑wall; FD eigen‑solve; WKB approximation; orthonormality & virial checks; convergence.
4. **Benchmarks** — deterministic metrics table (auto‑regenerates if missing).

## Local run
```bash
pip install -r requirements.txt
streamlit run app.py
```
